# Aviation Flashcards Web Application

A web-based flashcard application specifically designed for aviation study and learning.

## Project Structure

```
avion-cards-web/
├── public/
│   ├── index.html
│   ├── favicon.ico
│   └── assets/
│       └── placeholder.jpg
├── src/
│   ├── App.jsx
│   ├── main.jsx
│   ├── api/
│   ├── components/
│   ├── context/
│   ├── hooks/
│   ├── models/
│   ├── services/
│   ├── utils/
│   ├── routes/
│   └── styles/
└── ...
```

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```
